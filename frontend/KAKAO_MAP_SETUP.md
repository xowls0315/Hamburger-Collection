# 카카오맵 설정 가이드

## 📋 카카오맵 JavaScript API 키 설정

카카오맵을 사용하기 위해서는 카카오맵 JavaScript API 키가 필요합니다.

### 1. 카카오 개발자 콘솔에서 키 발급

1. [카카오 개발자 콘솔](https://developers.kakao.com/) 접속
2. **내 애플리케이션** → 애플리케이션 추가하기 (또는 기존 애플리케이션 선택)
3. **앱 설정** 메뉴 클릭
4. **플랫폼** 섹션에서 **"JavaScript SDK 도메인"** 찾기 ⭐ 필수
   - 도메인 입력 필드에 `http://localhost:3000` 입력 (개발 환경)
   - `+` 버튼 클릭하여 추가
   - 프로덕션 도메인도 별도로 추가 필요
   - **⚠️ 중요**: 도메인을 등록하지 않으면 JavaScript 키를 사용할 수 없습니다!
5. **앱 키** 섹션에서 **JavaScript 키** 복사

### 2. 환경 변수 설정

프론트엔드 루트에 `.env.local` 파일을 생성하고 다음을 추가:

```env
NEXT_PUBLIC_KAKAO_MAP_KEY=발급받은_JavaScript_키
```

**중요**: 
- `NEXT_PUBLIC_` 접두사가 필수입니다 (Next.js에서 클라이언트 사이드 환경 변수)
- `.env.local` 파일은 Git에 커밋하지 마세요 (`.gitignore`에 포함되어 있음)

### 3. 개발 서버 재시작

환경 변수를 추가한 후 **반드시 개발 서버를 재시작**해야 합니다:

```bash
# 개발 서버 중지 (Ctrl+C)
# 개발 서버 재시작
npm run dev
```

### 4. 확인 사항

- [ ] 카카오 개발자 콘솔에서 JavaScript 키 발급 완료
- [ ] JavaScript SDK 도메인에 `http://localhost:3000` 등록 완료
- [ ] `.env.local` 파일에 `NEXT_PUBLIC_KAKAO_MAP_KEY` 설정 완료
- [ ] 개발 서버 재시작 완료
- [ ] 브라우저 콘솔에서 "카카오맵 스크립트 로드 완료" 메시지 확인
- [ ] 브라우저에서 카카오맵이 정상적으로 표시되는지 확인

### 5. 문제 해결

#### 스크립트 로드 실패 시 확인 사항:

1. **API 키 확인**
   - `.env.local` 파일에 키가 올바르게 입력되었는지 확인
   - 키 앞뒤에 공백이나 따옴표가 없는지 확인

2. **도메인 등록 확인**
   - 카카오 개발자 콘솔 → 앱 설정 → 플랫폼
   - "JavaScript SDK 도메인"에 `http://localhost:3000`이 등록되어 있는지 확인
   - 등록 후 몇 분 정도 기다려야 적용될 수 있음

3. **브라우저 콘솔 확인**
   - 개발자 도구(F12) → Network 탭
   - `dapi.kakao.com/v2/maps/sdk.js` 요청이 실패하는지 확인
   - 에러 메시지 확인

4. **캐시 클리어**
   - 브라우저 캐시 삭제
   - 하드 리프레시 (Ctrl+Shift+R 또는 Cmd+Shift+R)

---

**참고**: 카카오맵 JavaScript API는 무료로 사용 가능합니다.
